Navigate to the Settings Page

You can use the menu link provided to easily and quickly access the settings tabs

Set you General Settings

Configure your settings to match your requirements. Below is an example of a general setup using the default magento platform settings. Click the to save the information.

Settings Descriptions

  • Enable/Disable — Enabling will add the extension functionality accross your platform
    • Enable or Disable the frontend reporting functionality

  • Allow Update Check — Enable to have the extension check for updates automatically (see here)
    • Check on the config page settings in the admin menu for message about update status of extension. Disable to stop checking.

Message Reports Settings

Select your settings for the message types to be recorded & to suit your requirements. Below is an example of a general setup using the default magento platform settings. Click the to save the information.

Message Reports Settings Descriptions

  • Success Reporting — Adding value will enabled recording of Success messages
    • Enables reporting of success messages
    • Enabled by Default

  • Error Reporting — Adding value will enabled recording of Error messages
    • Enables reporting of error messages
    • Enabled by Default

  • Notice Reporting — Adding value will enabled recording of Notice messages
    • Enables reporting of notice messages
    • Enabled by Default

  • Warning Reporting — Adding value will enabled recording of Warning messages
    • Enables reporting of warning messages
    • Enabled by Default

About Section

The About section will include the most latest information about the extension and any messages releating to extension updates that are available.
