About This Extension

Easily record multiple types of frontend user messages and manage the reports in your admin panel. This extension has been built specifically for the Magento Platform.

Key Features

  • Easily review error, success & warning messages from the frontend
  • Config Settings & Magento ACL
  • Extended custom magento grid
