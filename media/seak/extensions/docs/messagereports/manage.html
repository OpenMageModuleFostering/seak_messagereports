Navigate to the Message Reports Grid Page

You can use the menu link provided to easily and quickly access the grid page

Viewing Items in the Grid Page

Within the Grid page, you can view all of your recorded user messages with multiple columns showing associated information.

Review Report Item

You can use the review link on each item on the grid page. This will take you to the report review page where you can add or update/edit the information. Click the to save/update the information.


Field Types

  • Message Type
    • Type of message that has been recorded. Can either be Error, Success, Notice or Warning.

  • Message Content
    • Content body of the recorded message.

  • Message URL — Returned URL after submission/action
    • Displays the returned URL after message has been generated.

  • Message Action
    • Message Action route returned after message generation

  • Message Website
    • Website used to generate the message notification.

  • Message Store View
    • Store View used to generate the message notification

  • Customer
    • Displays the customer that generated the message. Returns Guest if generated by non logged in user

  • Admin Notes — Add any notes about the link (for admin purposes only)
    • Admins can add notes for other admin users to view or add your own info


Mass Deleting Links

You can delete links on the grid links page. In the left column, select the checkbox for each link you wish to delete.

Then select the delete field in the actions dropdown field on the right hand side. Click the to delete selected link(s).
